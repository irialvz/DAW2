Ejercicio 1. Investigación

¿En qué año se creó CSS y quién fue su creador? Fuente

Cascading Style Sheets o CSS se creó por la unión de dos propuestas de hoja de estilos,CHSS por Håkon Wium Lie y SSP por Bert Bos. Entre 1994 y 1995 se unieron para crear CSS.

¿Por qué surgió la necesidad de separar el contenido HTML del diseño CSS? Fuente

Surgió con la idea de separar la información y los datos en el HTML, y los aspectos y estilos para el CSS.Ademas de poder visualizar correctamente la informacion en diferentes dispositivos

¿Qué mejoras trajo consigo CSS3 respecto a versiones anteriores? Fuente

  • Permitieron maximizar la UX en los moviles con el uso de media queries. Nos permite añadir estilos según la medida del dispositivo.
  • Tambien se introdujeron propiedades para crear bordes y sombras

¿Qué ventajas ofrece CSS3 para diseñar interfaces modernas? Fuente

CSS3 mejora significativamente a CSS al ofrecer funciones avanzadas como diseño adaptable, animaciones y estructuras modulares. Estas mejoras hacen que el desarrollo web sea más rápido, eficiente y visualmente atractivo, lo que resalta la importancia de utilizar tecnologías web modernas para un mejor diseño.
